
# SecurityPartner

> Projet de l’unité d’enseignement Composants (CPS)

Ce projet vise à comprendre les principes d’une architecture à base de composants. L’objectif est d’implanter un prototype de système de gestion d’événements complexes. Ce type d’architecture logicielle devient populaire dans l’industrie depuis plus d’une décennie, mais il ne s’agit pas ici de construire un outil réaliste, simplement de comprendre certaines notions. De nombreuses simplifications seront nécessairement faites.
